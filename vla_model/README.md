## Обучение

- **Базовая модель:** SmolVLA (HuggingFace/lerobot)
- **Предобучение:** ~100k шагов на симуляционных данных (MuJoCo сцена с кубом и тарелкой)
- **Дообучение:** ~10k шагов на 15 реальных эпизодах телеоперации
- **Задача:** "Put cube on plate" - взять куб и поставить на тарелку

Веса модели (~1.7 ГБ) в репозиторий не включены из-за размера. Чекпоинты хранились в `smolvla_110k/` (файнтюн) и `smolvla_checkpoint/` (промежуточный).

## Что здесь есть

| Файл / папка | Описание |
|---|---|
| `run_smolvla_inference.py` | Инференс на реальном роботе: загрузка модели, захват с камер, предсказание action |
| `collect_data/` | Скрипты сбора датасета: телеоперация через leader arm или MuJoCo |
| `real-dataset/` | Первый реальный датасет (телеоперация на физическом роботе) |
| `real-dataset-2/` | Второй реальный датасет (дополнительные эпизоды) |

## Структура датасетов

Датасеты в формате LeRobot:
- `data/` - parquet-файлы с joint positions и actions
- `meta/` - метаданные (info.json, episodes, tasks)
- `videos/` - видео с камер (front + side)

## Инференс

```bash
python run_smolvla_inference.py \
    --policy-path path/to/smolvla_110k \
    --dataset-root ./real-dataset
```

Скрипт подключается к роботу (SO-101 follower на COM7), захватывает кадры с двух камер (front + side), прогоняет через модель и отправляет предсказанные joint angles на робот.

## Сбор данных

```bash
cd collect_data
python run.py
```

Параметры в `config.py`: порт leader arm (COM6), количество демонстраций, FPS, размер изображений.
